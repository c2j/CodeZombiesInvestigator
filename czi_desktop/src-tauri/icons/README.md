# Icons Directory

This directory should contain the application icons for different platforms:

- `32x32.png` - 32x32 pixel PNG icon
- `128x128.png` - 128x128 pixel PNG icon
- `128x128@2x.png` - 256x256 pixel PNG icon (retina)
- `icon.icns` - macOS icon file
- `icon.ico` - Windows icon file

For now, we'll create placeholder files. In a real project, you would add proper icon files here.
